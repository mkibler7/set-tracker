# Changelog

All notable changes to SetTrack will be documented in this file.

## v1.0.0

- Initial production release.
- Authentication:
  - register/login/logout
  - email verification
  - password reset
  - demo login
- Workout logging:
  - create/update workout sessions
  - add/remove exercises and sets
  - save and re-open sessions
- Exercises:
  - global exercise catalog
  - user exercise constraints
- Dashboard:
  - charts and summary metrics from logged data
- Deployment:
  - frontend on Vercel, backend on Render, database on MongoDB Atlas
